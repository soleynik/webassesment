shrpas2

prereqs

  1. have node.js installed and in your system path
  2. have java version 8 or above installed and in system path

building the project

in your command line inside the root folder type ### NOTE: you must run these commands as admin in windows run as administrator the terminal in linux preface the command with sudo, same with mac

./gradlew assemble

this will build all the necessary files together in the project

running the project

after the project is built you may run the project with

java -jar backend/build/libs/backend.jar

build and run

you may tie together the scripts to build and run the project. to make this easier create a build file to automate this process.

ex:

./gradlew assemble && java -jar backend/build/libs/backend.jar

testing

to test you need to call

./gradlew frontend/backend:test

you may remove frontend if you are testing only the backend,and backend if you are testing the frontend. you may keep both if you wish to run tests for both projects.
